# sql-typeck

## Setup

1. Create virtual environment: `python3 -m venv venv`
2. Activate virtual environment: `source venv/bin/activate`
3. Install requirements: `pip3 install -r requirements.txt`
4. Run unit tests: `python3 -m unittest`
